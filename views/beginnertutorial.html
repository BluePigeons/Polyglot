Tutorial

Welcome to the start of your annotating adventure! Wooooo!

So we appear to have found a lovely picture here that appears to have some writing in it! Hooray! But no one has marked on the picture where the words are. Use the marker buttons (currently highlighted in yellow for you) to select the words on the page.

Well done on adding your first marker!

Once a marker is added to a picture you can then start to add information about it. Click on the marker to see that a menu opens to let you do things with either the transcriptions (writing what you see on the page) or the translations for that section. Select the transcription!

No one has written a transcription for this marker so it looks like you'll have to add one yourself - write in the box what you see written in the marker and click the submit button to see what happens next!

So here is the document again but this time someone has already made the marker and if you click on it and select transcription you'll see that they have already added a transcription too.

Now unlike before you'll see that you cannot add a new transcription - but what if you disagree with what was written? Can you see the word "gastropub" where it should be "gastrobob" instead? Select the incorrect word in the editor box! You'll see another popup menu but this time in the editor. You can do the same thing as before and view the transcription.

Of course this time you'll notice that we already have one existing transcription with the word "gastropub" but if you select the "add new" button or scroll through the editor panes then you will get to the same add new page as before. In here add the corrected word of "gastrobob" and submit!

Hoorah! We corrected it! ...But did we?
