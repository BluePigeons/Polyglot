About Us

So the Polyanno project was developed at the University of Edinburgh Library and Information Services group to crowd source the transcription and translation of the collections.

  • "Polyglot"

    - The original overall project of developing the website for the University of Edinburgh - you can see the code to develop this website at the Github repo here. This uses the projects below as well as a script to select and filter images from the University of Edinburgh LUNA database to present within Polyanno framework and additional pages.

  • Polyanno

    - The Javascript framework developed to allow implementation of crowd sourcing transcriptions and translation of IIIF images using Leaflet, Dragon_drop and AllTheUnicode. Git repo here.

  • AllTheUnicode

    - The Javascript framework to implement multiple different keyboards and IMEs within a web page to allow inputs across the Unicode spectrum. Git repo here.

  • Dragon_drop

    - The Javascript framework to implement a draggable, resizeable, minimizeable and closeable windows UI within the browser web page. Git repo here.

  • Polyanno Storage

    - The Node and Express package to allow MongoDB storage of annotations in Open Annotation Model format of Polyanno websites. Available both as code on Git Repo here but also as an NPM package here.
